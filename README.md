# `.github`

This repository holds organisation-level configuration and the profile content shown on The Longhand Archive's [GitHub organisation page](https://github.com/longhandarchive).

Most visitors do not need to read anything here. The content that matters is rendered automatically on the organisation page and on individual repositories.

## What lives here

**`profile/README.md`** — the content rendered on the organisation page at `github.com/longhandarchive`. This is the public-facing introduction to The Longhand Archive.
